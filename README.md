# contacts_app
 text based contacts app (saves contacts to a sqlite3 .db file)
 
 1. Add new Contact.
 2. DELETE Contact. 
 3. Display all contacts. 
 4. Search contacts. 
 5. Quit contacts app

python libraries used:
sqlite3
pandas
